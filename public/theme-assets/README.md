# Dark Neon Sustainability Theme Assets

Generated for the digital sustainability / eco-tech theme (dark background, electric green and cyan accents).

| File | Use |
|------|-----|
| `hero-digital-sustainability.png` | Hero section background (dark, glowing green/cyan network + plant). Use as full-bleed background with overlay for headline + CTA. |
| `pattern-circuit-dark.png` | Tileable section background. Dark blue-teal with subtle circuit traces. Use as `background-image: url(/theme-assets/pattern-circuit-dark.png)` with `background-repeat: repeat`. |
| `wireframe-plant-transparent.png` | Decorative graphic (transparent). Wireframe plant in green/blue. Use in hero or section as floating illustration. |
| `favicon-msrd.png` | App icon / favicon. Green "M" on dark. Reference in `<link rel="icon" href="/theme-assets/favicon-msrd.png" />`. |

When implementing the dark theme, update:
- Hero: `backgroundImage: 'url(/theme-assets/hero-digital-sustainability.png)'`
- Section backgrounds: add pattern where desired
- Favicon: already set in index.html
